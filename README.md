# Data Science Projects

Portfolio de proyectos de ciencia de datos desarrollados con Python, orientados al análisis exploratorio, visualización, modelado predictivo y comunicación de resultados.

## Projects

### 01 - Electric Power Forecasting Argentina

Forecasting de capacidad eléctrica instalada en Argentina utilizando datos públicos del sector energético y modelos de series temporales.

**Tech stack:** Python, Pandas, NumPy, Matplotlib, Statsmodels, Scikit-learn  
**Model:** SARIMAX  
**Methodology:** CRISP-DM
